# Screenshots Guide

These screenshots are needed for the README, website, LinkedIn post, and Show HN.

---

## Required screenshots

### 1. Studio home screen (Setup Panel)
- State: fresh load, setup panel visible
- Shows: track grid, mode grid, provider selector
- Capture: full browser window at 1440x900

### 2. Active chat interview
- State: session active, 4-6 messages visible
- Shows: interviewer question, candidate answer, typing indicator
- Capture: center chat column, or full 3-column layout

### 3. Avatar coach — speaking state
- State: avatar in "speaking" state (mouth animation)
- Shows: SVG avatar, state label "Speaking...", tip box
- Capture: right panel only, or full layout

### 4. Avatar coach — thinking state
- State: avatar in "thinking" state (dots animation)
- Capture: right panel

### 5. Session sidebar with history
- State: 3-4 past sessions listed
- Shows: track names, mode, status badges, dates
- Capture: left sidebar only

### 6. Scorecard panel
- State: finished session, scorecard visible
- Shows: overall score, hire signal badge, dimension bars
- Capture: center column

### 7. Readiness report
- State: report generated after finishing session
- Shows: readiness score, hire signal, weakness map, top actions
- Capture: full report panel

### 8. STAR story bank
- State: 2-3 stories added
- Shows: story cards with category badges, tags
- Capture: full feature view

### 9. Pricing page
- State: all 5 plan cards visible
- Shows: Free/Pro/Premium/Lifetime/Team cards
- Capture: full page

---

## Tips

- Use the mock provider — no API key needed
- Dark mode is default — screenshots look best on a dark background
- 1440x900 or 1280x800 browser window
- DevTools zoom: 100%
- Chrome or Edge recommended for best font rendering
